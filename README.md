# CaliBot 🤖

CaliBot is a scalable multi-server Discord bot built with Python, PostgreSQL, and a modular architecture.

It is designed as a production-ready project, featuring:
- Multi-server (guild-based) configuration
- XP and leveling system
- Database persistence with PostgreSQL
- Clean and maintainable structure
- Future integration with a web dashboard (React + FastAPI)

This project is part of my backend development portfolio.
